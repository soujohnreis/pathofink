Path of Ink — Licenses & Credits

Updated: June 15, 2026

Path of Ink is built using the Defold Game Engine and several open-source libraries. The soundtrack and ambient musical elements are generated using Google Flow Music. Below is the copyright and licensing information for all third-party components.

Soundtrack Credits

The original soundtrack, loop beds, and ambient audio components for Path of Ink were generated using Google Flow Music (powered by Google DeepMind's Lyria 3 Pro music model). They are used under a valid commercial use license tier.
